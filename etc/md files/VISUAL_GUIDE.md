## 🎯 Quick Reference

### To Add Content:
1. Admin → ಅಧ್ಯಾಯಗಳು → Fill form → ಉಳಿಸಿ
2. Admin → ಪರೀಕ್ಷೆಗಳು → Select chapter → Add questions → ಪರೀಕ್ಷೆ ಉಳಿಸಿ

### To Learn:
1. Students → Click chapter → Read → ಪರೀಕ್ಷೆ ಪ್ರಾರಂಭಿಸಿ → Answer → ಪರೀಕ್ಷೆ ಸಲ್ಲಿಸಿ

### To Backup:
Copy these files:
- `backend/data/chapters.json`
- `backend/data/quizzes.json`
